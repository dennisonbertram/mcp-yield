## Platform-Specific Requirements

### Dev.to
- **Size**: 1000x420px (or ratio 1000:420)
- **Format**: PNG or JPEG
- **File size**: Under 500KB recommended
- **Text**: Readable at desktop and mobile sizes

### Medium
- **Size**: 1500x750px minimum (will crop to 2:1 aspect)
- **Safe zone**: Keep important elements in center 1400x700px
- **Format**: JPEG or PNG
- **File size**: Under 5MB
- **Note**: Medium crops aggressively on mobile, test your design

### LinkedIn
- **Size**: 1200x627px (exact for article)
- **Aspect ratio**: 1.91:1 (critical for proper display)
- **Format**: JPEG or PNG
- **File size**: Under 5MB
- **Text**: Professional fonts, large enough to read on mobile

### Hashnode
- **Size**: 1600x840px recommended
- **Format**: PNG or JPEG
- **File size**: Under 1MB recommended
- **Note**: Hashnode shows full image, no aggressive cropping

## DIY Design Tools

### Free Tools
- **Canva**: Templates for all platforms, easy text overlay
- **Figma**: More control, free tier sufficient
- **Photopea**: Free Photoshop alternative in browser
- **GIMP**: Open-source image editor

### Quick Tips
1. **Typography**: Use maximum 2 fonts (1 for title, 1 for body)
2. **Contrast**: Ensure text readable against background (use dark overlay on images)
3. **Simplicity**: Better to be simple and clear than complex and confusing
4. **Consistency**: Use same color palette across all images
5. **Test**: View at thumbnail size before publishing

## AI Image Generator Settings

### Midjourney
```
/imagine [prompt] --ar 1000:420 --v 6 --style raw
```
- Use `--ar` for aspect ratio
- `--style raw` for more literal interpretation
- `--v 6` for latest version
- Add `--no text` if you want to add text yourself

### DALL-E 3 (ChatGPT)
```
"Create a [style] blog header image, [description],
dimensions should be landscape format suitable for [platform],
modern and professional style"
```
- DALL-E 3 handles text better but still verify spelling
- Request "high contrast" for readability
- Specify "professional" or "corporate" for business content

### Stable Diffusion
```
[prompt], professional blog header, clean design,
high contrast, readable text, modern aesthetic,
trending on dribbble, award winning design
```
- Add quality tags: "trending on dribbble", "award winning"
- Negative prompt: "cluttered, busy, illegible text, low contrast"
- Use img2img to refine outputs

## Testing Your Images

Before publishing:

1. **Desktop view**: Check full image looks good
2. **Mobile view**: Check mobile crop (usually center or top)
3. **Thumbnail view**: Ensure readable at 200px wide
4. **Share preview**: Use Facebook Debug Tool to see how it renders in shares
5. **Contrast check**: Use WebAIM contrast checker for text readability

**Tools**:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/

## Quick Checklist

Before finalizing any cover image:

- [ ] Correct dimensions for platform
- [ ] File size under platform limit
- [ ] Text readable at thumbnail size
- [ ] High contrast (text vs background)
- [ ] Professional/appropriate for content tone
- [ ] Consistent with brand colors (if established)
- [ ] No copyrighted elements
- [ ] Tested on mobile view
- [ ] No critical info in crop zones
- [ ] Alt text planned for accessibility
